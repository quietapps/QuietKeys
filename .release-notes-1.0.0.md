Initial release of Quiet Keys — realistic mechanical keyboard switch sounds for every keystroke and mouse click, fully offline.

## Highlights

- **22 switch profiles** across 13 brands (IQUNIX, Lofree, Akko, Keychron, Aflion, Durock, Gateron, NovelKeys, Drop, Kailh, IBM, Topre, Alps) plus a quirky Lizard — every sample synthesized from parametric switch models
- **Ultra-low latency** — lock-free audio engine, 128-frame CoreAudio buffer, all samples preloaded
- **Spatial audio** — keys pan by their physical position on the keyboard
- **Distinct press and release sounds** with per-key-class samples and round-robin variants
- **Mouse click sounds** for left, right, and middle buttons
- **Reactive visualizer** — a floating mini keyboard that lights up as you type
- **Typing test** with WPM, accuracy, and timer
- **Custom switch profiles** — drop-in sample folders, no rebuild needed
- **Fully private** — listen-only event tap, no data collected, no network access

## Install

```sh
brew tap quietapps/quietkeys
brew install --cask quietkeys
```

Or download `QuietKeys-1.0.0.zip` below, drag **Quiet Keys.app** to `/Applications`, and run `xattr -cr "/Applications/Quiet Keys.app"` once.

Requires macOS 13 Ventura or later. See the [CHANGELOG](https://github.com/quietapps/QuietKeys/blob/main/CHANGELOG.md) for full notes.
