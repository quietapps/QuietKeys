Visualizer grows up: labels, drag, resize.

## Added

- **Close button** — hover the visualizer and a small ⨯ appears in the top-right corner; click to hide it without opening the menu
- **Key labels** — optional legends on the mini keyboard (Settings → Visualizer → *Show key labels*)
- **Drag to move** — hover, then drag the visualizer anywhere; the position is remembered across launches
- **Drag to resize** — a corner handle scales the visualizer from its default size up to 2.5×, aspect locked
- **Reset position and size** — one button in Settings → Visualizer restores the defaults

The visualizer stays fully click-through until the cursor is over it, so it never steals clicks while you work.

## Install

```sh
brew tap quietapps/quietkeys
brew install --cask quietkeys
```

Or download `QuietKeys-1.0.2.zip` below, drag **Quiet Keys.app** to `/Applications`, and run `xattr -cr "/Applications/Quiet Keys.app"` once.

Requires macOS 13 Ventura or later. See the [CHANGELOG](https://github.com/quietapps/QuietKeys/blob/main/CHANGELOG.md) for full notes.
